analysis module¶

The analysis module contains the components of the inference related to SatGen:

  1. The SatGenData class acts as a wrapper around the processed SatGen output, and holds the direct sampling of \(f_\mathrm{pred}(\boldsymbol{x},\,\boldsymbol{\theta})\) (i.e., the full parameter information of each SatGen satellite) as well as the interpolated \(f_\mathrm{pred}(\boldsymbol{\theta})\) distributions.

  2. The Weights class combines these with observed values to compute the weight applied to each SatGen satellite for a given choice of \(\boldsymbol{\theta}\).

Classes¶

Provides weights for SatGen output such that their distribution can approximate observed probability distributions.
